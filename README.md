# Jopa

**Jopa**  `[ˈʐopə]`  is  a  very  small,  but  powerful  static  website
generator  writtent  in  bash.  It  uses  a  couple  of  conventions  to
make  writing  stuff fun  and  easy.  Read  more on  the  [documentation
website][docs].
